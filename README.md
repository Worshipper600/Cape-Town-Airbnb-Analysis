# Cape-Town-Airbnb-Analysis
## Software Development for Data Science
## Glasgow Caledonian University

This project analyzes the CapeTown Airbnb Dataset, with a focusing on data cleaning, wrangling, and exploratory analysis. This project aims to prepare the dataset for further analysis and proposes three potential research questions based on findings from the data exploration. Dataset merging, dealing with missing values and duplicates, and investigating correlations between important variables are all key operations. <br><br>
